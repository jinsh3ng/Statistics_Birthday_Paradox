# Birthday Paradox Exploration

I came across the Birthday Paradox while working through some probability questions, and it immediately caught my attention.

The question is simple: in a random group of 30 people, what is the probability that everyone has different birthdays?

The answer, roughly 30%, felt counterintuitive at first. In other words, there is about a 70% chance that at least two people share the same birthday. With 365 possible days, it seems surprising that such a small group would produce a match so often. Yet both the mathematics and simulations suggest otherwise.

This project is a small exploration driven by curiosity. Nothing groundbreaking, but just a simple attempt to verify the theory using code and, later, real-world data.
